Mis Proyectos

Proyecto 1: Cotizacion de transporte privado RIVOGUE

Un programa de escritorio que esta pensando en una empresa de transportes privados en el cual puedes registrar el viaje que el cliente pide.

Esta conectado a una base de datos para guardar la informacion del viaje de los clientes.

Las tecnologias usadas fueron java y mysql workbench

Proyecto 2: Punto de venta con prediccion de ventas.

Una pagina web que esta destinado a un local que vende botanas y dulces en el cual se le hizo un punto de venta pero con la ventaja de que tiene una prediccion de ventas semanales totales.

Las tecnologias a usar fueron html,css,javascript y boostrap acompañado de un sistema inteligente basado en una rama del Machine Learnin que es regresion lineal la cual tiene una rama sobre regresion polinomial ya que se desarrollaba mejor la prediccion con ese tipo de regresion.

Proyecto 3: Juego Run Bruja Run

Juego basado en los primeros juegos hechos en la computadora tipo arcade con la idea principal era idealizarse con la logica de como hacer videojuegos y al final se logro un juego donde una bruja dispara bolas de energia hacia los esqueletos que son sus enemigos, al momento de matar ciertos enemgios sale el jefe final que le ganas disparandole 5 veces, la bruja solo tiene 3 vidas.

Las tecnologias usadas fueron python y pygame junto con cocos2d.

Proyecto 4: Juego -BrincaFrijol-

Un juego hecho con unity con basado en algun juego ya existente pero que no sea igual.

El juego fue basado en Geometry Dash y ahora es un frijol que no debe de caer o golpear los objetos pero esta vez hay enemigos que son tortillas que si los tocas te hacen taco de frijol.
